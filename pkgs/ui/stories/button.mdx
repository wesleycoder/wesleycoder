## Usage

### Primary

A button can be of primary importance.

<Canvas
  withToolbar
  sourceState='shown'
  layout='centered'
  of={ButtonStories.Primary}
  additionalActions={[
    {
      title: 'Open in GitHub',
      onClick: () => {
        window.open(
          'https://github.com/storybookjs/storybook/blob/next/code/ui/blocks/src/examples/Button.stories.tsx',
          '_blank'
        );
      },
    }
  ]}
/>
